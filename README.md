# Calculator

This repository is for 9spokes code challenge.

## Clone the project

```
$ git clone https://github.com/zan768616253/coding-challenge.git
```

## Install dependencies

```
$ cd coding-challenge
$ go mod tidy
```

## Build Project

```
$ go build cmd/main.go
```

## Run Project

```
$ ./main
```
## Run Unit Test

```
$ ginkgo test
```
